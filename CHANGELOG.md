# Changelog

All notable changes to public releases are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and [Semantic Versioning](https://semver.org/).

## [Unreleased]

## [2.1.0] - 2026-03-29

### Changed
- Windows is now the primary and only actively updated platform
- Public release documentation was rewritten around the native Windows desktop runtime
- Download guidance now clearly separates the active Windows line from archived macOS/Linux builds

### Added
- Windows x64 release asset: `Hotmail.Multi-Checker-2.1.0.exe`
- Updated public release notes and release process for Windows-first publishing

### Deprecated
- Future macOS and Linux feature updates

### Notes
- macOS and Linux remain available only as archived `1.1.0` downloads

## [1.1.0] - 2026-03-25

### Added
- Windows x64 release asset: `Hotmail.Multi-Checker-1.1.0.exe`
- macOS and Linux `1.1.0` release line
- Custom module search support
- CPM stat, proxy management page with format selector, and live proxy testing
- Capture category toggles, multi-country filter, and profile enrichment

## [1.0.0] - 2026-03-22

### Added
- Initial public release
- Multi-threaded account checking engine
- Linked-service capture for PSN, Epic Games, and Instagram
- Proxy support with rotation
- Real-time dashboard, logs, and hit management
- License activation flow
